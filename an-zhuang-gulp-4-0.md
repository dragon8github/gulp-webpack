### 安装gulp 4.0

想体验4.0只有通过github安装，执行以下两条命令即可在本地畅爽地使用gulp 4.0了。

```js
$ npm install gulpjs/gulp#4.0 -g
```

### Gulp 4.0 两个核心的变化：

1. `gulp.task` 的变化；
2. 增加了`gulp.series`和`gulp.parallel`。
